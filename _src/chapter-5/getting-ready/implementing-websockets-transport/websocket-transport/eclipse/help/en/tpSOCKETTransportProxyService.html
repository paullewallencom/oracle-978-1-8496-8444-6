Socket Transport Configuration Page (Proxy Services)

The socket transport is an example of a custom transport. It communicates with other services using TPC/IP. The protocol is simple by design, letting you focus on the core implementation details of the transport SDK used in the sample to help you build your own transports. For details on building your own transports, see the Oracle Service Bus Transport SDK User Guide.

In the IDE, when you select the socket transport for a proxy or business service, the editor provides a SOCKET page for configuring the transport. The fields in the editor are mainly for illustration purposes, showing you how you can provide user interface controls to let users configure your custom transports.

This help topic also illustrates how you can incorporate your own help content for your transports in the IDE through context-sensitive help (pressing F1 in the transport configuration page) and in the IDE help contents (Help > Help Contents).

You can also deploy and use the socket transport in the Oracle Service Bus console, as described in the Oracle Service Bus Transport SDK User Guide.
